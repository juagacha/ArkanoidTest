# ArkanoidTest



Es un juego tipo arcade construida para ejecutarse en window o web, se mueve el paddle por medio de las teclas ad o flechas, se da inicio con espacio. En este caso se rompen platos, existen tres tipos el basico, el rodeado por una caja de cristal, y el rodeado por una caja de ladrillo, los cuales se rompen por la cantidad de golpes dados, el normal 1, el de cristal 2 y el de ladrillo 3. Posee dos power up, uno amarillo que suma una vida y otro rojo que suma puntos.
